# Time Series Models

## Models in this section
Empirical time series analysis of macroeconomic data using
FRED and Stock-Watson methods.

## Common patterns
- Data loading from FRED-MD dataset (CSV)
- Factor models and principal components
- Forecasting and impulse responses from reduced-form VARs

## Key economics
- Macroeconomic monitoring with high-dimensional data
- Stock-Watson diffusion index methodology
- Business cycle measurement and dating
